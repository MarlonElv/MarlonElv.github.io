Incorrect Mod Rail Qty Led to Impossible MH/U Target
Estimating incorrectly bid the mod rail quantities, producing a budgeted MH/U that was mathematically impossible to achieve on-site. The field team was benchmarked against a number that no level of productivity could reach.
Budget Impact
No Phase Code for Trash Removal & Remediation
No phase code was established for activities such as trash removal or remediation. Hours consumed by these tasks were absorbed into other phase codes, distorting productivity data across the project.
Phase Code Gap
Undefined Phase Code — Install Pile Caps/BHA
Phase PHASE: 01048100M32 — INSTALL PILE CAPS/BHA lacks a clear definition and the quantities do not align with any identifiable field activity, creating a reconciliation gap in the cost report.

2-Week Delay Cascaded Across All Mechanical Scopes
The initial Gerdau fabrication delay on 8/11 pushed the entire pile install schedule by two weeks, directly delaying tracker pile install (10/1 E2, 10/8 E1), racking install (10/29), and module install (11/9). Every downstream mechanical scope absorbed compounding schedule pressure.
Schedule Impact
Torque Check Remediation Cost — 220–300 Hours
Zone 8/9 torque checks ran from 1/23 to 1/27/2026 accumulating 210–300 labor hours with additional OT (166 hrs) and DT (40 hrs). This cost was a direct consequence of pile installation quality issues downstream of the procurement failure.
Cost Impact
3,542 Pile Refusals Identified — Rework Plan Pending
PRE survey data processed 4/16/2026 revealed 3,542 pile refusals across both sites. No resolution or rework plan has been finalized. This represents a significant outstanding risk to structural completion and final inspections.
Open Risk
Equipment Requests Not Processed Until 11/4
Multiple equipment requests (UTVs, forklifts) submitted earlier were never processed. The team only discovered this on 11/4/2025 after following up and confirming fleet had not received them. Contributed to ramp-up inefficiencies during peak install.

Site UXO Risk Was Not Pre-Surveyed Comprehensively
Four separate UXO discovery events across different zones throughout the project suggests the pre-construction UXO survey did not cover all areas comprehensively, or phased clearing was not coordinated with civil scope sequencing.
Planning Gap
IPX Backfill Protocol Caused Recurring Daily Stoppages
IPX stopped backfill operations daily throughout the project — failing to understand that trench cleaning occurs simultaneously with cable shading. This process misalignment caused recurring, preventable stoppages beyond the formal UXO hold events.
Process Gap
Contingency Float Should Be Built Into Civil Schedule
Given the known UXO risk profile of this site (4 events, multiple zones), future projects on similar sites should include explicit UXO contingency float in the civil CPM schedule — minimum 2 weeks per zone based on observed impact.

Lessons Learned & Suggestions
LESSONS
Key Takeaways
01
Estimating Incorrectly Bid Mod Rail Quantities
This led to a high budgeted MH/U — a target impossible to hit. The result was field teams being measured against an unachievable baseline from day one.
02
No Phase Code Established for Remediation Activities
Activities such as trash removal and remediation had no designated phase code, causing hours to bleed into unrelated codes and distorting all productivity reporting.
03
Undefined Phase Codes with Misaligned Quantities
Phase codes such as PHASE: 01048100M32 — INSTALL PILE CAPS/BHA lack clear definitions and the quantities don't align with any field activity, creating reconciliation problems in cost reporting.
Lessons Learned & Suggestions
01
Estimating Must Work Closer with Experienced Field Teams
Estimators should partner with experienced field superintendents and PMs during bid development — particularly for novel system types or repeat vendors where historical productivity data exists.
02
Document Estimating Process so Field Teams Can Interpret Phase Codes
A written estimating narrative should accompany every project phase code list so on-site teams can correctly interpret scope boundaries, unit of measure, and productivity assumptions.
03
There Needs to Be Better Kick-Offs
A more rigorous pre-construction kick-off process — including subcontractor alignment, phase code walkthroughs, procurement lead-time review, and UXO/civil risk identification — would prevent the cascading issues observed on this project.
